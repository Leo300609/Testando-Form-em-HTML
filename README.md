# 🚗 Venda seu Carro - Formulário de Cadastro

Este projeto consiste em uma interface de cadastro para venda de veículos, desenvolvida com o objetivo de praticar a estruturação de formulários complexos em HTML5 e a estilização avançada com CSS3.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica, uso de diferentes tipos de `input` (text, number, date, radio, checkbox, file) e atributos de validação.
* **CSS3:** Reset de estilos, layout responsivo com containers centralizados, efeitos de transição (`transition`) e estilização de formulários modernos.

## 🧠 O que foi desenvolvido:
* **Estrutura de Formulário:** Implementação de campos obrigatórios utilizando o atributo `required`.
* **Componentes de UI:** * Seleção de câmbio via botões `radio`.
    * Lista de opcionais utilizando `checkbox`.
    * Área de upload de imagens permitindo múltiplos arquivos (`multiple`).
* **Design e UX:**
    * Layout com cabeçalho destacado e formulário sobreposto (efeito de profundidade).
    * Feedback visual no botão de envio com efeito de `hover`.
    * Tipografia e paleta de cores focadas na legibilidade.

## ⚙️ Como visualizar
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
